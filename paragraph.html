Paragraph Tag

In html, paragraph tags are used to define a block of text as a paragraph and it’s a block level element which means it always starts in a new line.
Also html tags are not case sensitive .Though, developers prefer to use lowercase when writing html code because it will not be looking good when we use mix of uppercase and lowercase.

What is the difference between “tags and elements” in html ?

Tags

Tags are the beginning and finish parts of an HTML element. Whatever written inside < and > are called tag.
Html tags are nothing but it’s a keywords and tells the browser how to display the contents on a webpage. Each and every tag has a specific name. Tag start with less than symbol < and end with grater than symbol > .

There are two type of tags

1.opening tag/strating tag

2.closing tag/ending tag . Ending tag has forward slash flowed by less than symbol

Some tags are self-closed tags which means that they do not have closing tag. for eg:-< img/> -- image tag. Some are empty tags for eg:-br - break tag, hr - horizontal line tag

Elements

An element is an individual component of an HTML file and it consists of both the starting and ending tags with content in between.

Some element are block level elements means they always start in a new line.

for Eg:- paragraph , form and ect.

Some elements are inline elements. Inline element width is based on how much space that particular element needs.

for Eg:- image, button
